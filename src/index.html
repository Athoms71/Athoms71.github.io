Phishing Awareness

Statut : Terminé


Description : Développement d'un projet de sensibilisation au phishing pour promouvoir l'éducation à la cybersécurité. Ce projet utilise des scénarios réels générés par IA pour former les utilisateurs. L'objectif est de créer une plateforme clé en main pour le client qui simule des attaques de phishing afin d'aider les utilisateurs à reconnaître et à éviter ces menaces en ligne. Le projet a été réalisé en collaboration avec deux autres étudiants et notre professeur référent, et a permis de développer des compétences en développement web, en IA et en sensibilisation à la cybersécurité.

PolyHome

Statut : En cours


Description : Application native Android pour prendre en main les fonctionnalités de base autour du système afin de piloter une simulation Web d'une maison domotique. Ce projet permet d'explorer les concepts de IoT et de développement mobile, tout en offrant une expérience pratique de l'intégration entre une application mobile et une simulation web.

Marty The Robot

Statut : Terminé


Description : Projet de robotique éducative utilisant une plateforme de développement pour créer un robot interactif. Le projet a été réalisé en collaboration avec une équipe d'étudiants passionnés.
